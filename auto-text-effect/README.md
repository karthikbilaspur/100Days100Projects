# Inspirational Quote Typer

A simple and elegant web application that displays inspirational quotes with a classic typewriter effect. Users can control the typing speed, reset it, and navigate through a collection of motivating quotes.

## ✨ Features

* Typewriter Effect: Quotes are revealed character by character, simulating a typewriter.
* Adjustable Typing Speed: Control how fast the quotes are typed out using an input slider.
* Pause/Play Functionality: Temporarily halt or resume the typing animation at any point.
* Next Quote: Easily advance to the next inspirational quote in the collection.
* Reset Speed: Quickly return the typing speed to its default setting.
* Quote Repetition Prevention: The application intelligently shuffles quotes to ensure a fresh experience without immediate repeats.
* Responsive Design: Optimized for various screen sizes, from desktops to mobile devices.
* Subtle Animation: A calming background animation enhances the user experience.


## 🛠️ Technologies Used

* **HTML5:** Structure of the web page.
* **CSS3:** Styling and animations (including a custom Google Font, Roboto).
* **JavaScript (ES6+):** Core logic for the typing effect, speed control, quote management, and user interactions.

## 📖 Code Structure

The project consists of three main files:

* `index.html`: The main HTML file that defines the structure and content of the web page.
* `style.css`: Contains all the CSS rules for styling the elements and creating animations.
* `script.js`: Implements the interactive logic, including the typewriter effect, event listeners for controls, and quote management.
