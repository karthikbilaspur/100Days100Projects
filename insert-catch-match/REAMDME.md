# Catch The Insect" game

The game has been significantly enhanced to offer more engaging gameplay and a polished user experience. Key improvements include:

Difficulty Levels: Players can now choose between Easy, Normal, and Hard difficulties, which dynamically adjust the game's duration and how frequently insects spawn.
Lives System: A new "Lives" mechanic has been introduced. Players start with a set number of lives, losing one if an insect is missed (either by not clicking it before it disappears or by clicking outside an insect). The game ends when all lives are lost.
Enhanced Game Over Screen: Instead of a simple message, there's now a dedicated "Game Over" screen displaying the player's final score, the all-time high score, and options to "Play Again" (restarting with the same difficulty) or "Back to Main Menu" (to choose a new difficulty).
Smoother Gameplay Flow:
The "Play Game" button is now enabled only after a difficulty is selected.
Background music starts when the game begins and stops on game over.
Insects now have a "move away" animation, which, if not caught, triggers a life loss.
The game UI now clearly displays time, score, high score, and remaining lives.
