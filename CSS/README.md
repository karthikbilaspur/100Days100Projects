# 🚀 Overview

This CSS collection contains multiple front-end mini projects designed to strengthen practical web development skills through hands-on experimentation.

The projects cover:

* CSS animations and transitions
* Responsive layouts
* Flexbox and Grid systems
* Media queries
* Interactive UI components
* CSS loaders and preloaders
* Responsive dashboards
* Image galleries
* Small JavaScript integrations
* Accessibility enhancements

Each project is self-contained and includes its own source files.

# ⚡ Getting Started

Follow these steps to run any project locally.

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/100Days100Projects.git
```

## 2️⃣ Navigate to the CSS Folder

```bash
cd 100Days100Projects/CSS
```

## 3️⃣ Open Any Project

Example:

```bash
cd CSSLoader
```

## 4️⃣ Run the Project

Simply open the `index.html` file in your browser.

You can also use the VS Code Live Server extension for a better development experience.
