# Simple Blog Project

A simple blog project created using HTML, CSS, and JavaScript. The project features a responsive design, a search form, and a sidebar with categories.
Features
Responsive design using CSS media queries
Search form with JavaScript event listener
Sidebar with categories
Blog posts table with hover effects
Technologies Used
HTML5
CSS3
JavaScript (ES6+)

Open the index.html file in a web browser to view the project
Usage
Use the search form to search for blog posts.
Click on a category in the sidebar to view related blog posts.
Hover over a blog post in the table to view its details.
