Soundboard
A web-based soundboard application that allows users to play different sounds.
Features
Play different sounds, such as applause, boos, and cheering
Keyboard navigation using arrow keys and Enter key
Loading state indicator for sounds
Accessible design with ARIA attributes
Responsive design using CSS Grid and Flexbox
Technologies Used
HTML5
CSS3 (using Sass for customization)
JavaScript (using modern features and error handling)
Installation
Clone the repository using git clone https://github.com/your-username/soundboard.git.
Open the index.html file in a web browser to use the soundboard.
Usage
Click on a sound button to play the corresponding sound.
Use the arrow keys to navigate between sound buttons.
Press the Enter key to play the currently selected sound.
