What is this?

About This Project

This "Interactive Ripple Hoverboard" is a creative coding experiment designed to explore dynamic visual feedback on web pages. It demonstrates how simple HTML elements, combined with CSS transitions and JavaScript logic, can create an engaging and artistic user experience.

The core idea is to make each square in a grid react to your mouse by changing color and, importantly, triggering a "ripple" effect that subtly illuminates its neighbors. This creates a fluid, organic visual flow as you move your cursor.

Inspiration

Inspired by classic interactive grid effects and the beauty of cascading animations, this project aims to provide a simple yet captivating interaction. It's a testament to the power of front-end technologies to build delightful web experiences.
