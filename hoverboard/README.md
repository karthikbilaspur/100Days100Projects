# HoverBoard

A basic interactive hoverboard to a more complete and dynamic web application. Key improvements include:

Enhanced HTML Structure: Implemented semantic HTML5 (header main, footer, section), comprehensive SEO and social media meta tags, and added structure for modals.
Advanced CSS Styling: Introduced a CSS variable-based dark/light mode theme, integrated Font Awesome for icons, and added styling for interactive pop-up modals, header, footer, and responsive layouts.
Dynamic JavaScript Functionality: Expanded the interactive ripple effect, implemented client-side theme toggling with localStorage persistence, managed modal open/close logic, and added user-configurable settings (ripple radius, fade duration, board size) with localStorage saving and dynamic board re-initialization.
